# Демонстрационное приложение интеллектуального сервиса для формирования глобального инвестиционного портфеля

## 1. Описание

Демонстрационное приложение на Angular, предназначенное для демонстрации работы сервиса построения и оптимизации глобальных инвестиционных портфелей. Сервис использует интеллектуальный диалоговый интерфейс на базе больших языковых моделей (LLM) для взаимодействия с пользователем на естественном языке.

---

## 2. Функциональные возможности

Приложение демонстрирует цикл работы: от текстового запроса пользователя до получения и визуализации инвестиционного портфеля.

1.  **Построение портфеля:** Генерация структуры портфеля на основе текстового запроса.
2.  **Оптимизация портфеля:** Оптимизация структуры портфеля на основе текстового запроса.
3.  **Просмотр сохраненных портфелей:** Доступ к истории ранее созданных или оптимизированных портфелей.
4.  **Интеллектуальный диалог:** Взаимодействие с системой через чат-интерфейс.

---

## 3. Начало работы

### Предварительные требования
*   Node.js (рекомендуется LTS версия)
*   Angular CLI (`npm install -g @angular/cli`)

### Установка
1. Клонируйте репозиторий.
2. Установите зависимости:
   ```bash
   npm install
   ```

### Запуск
Для запуска приложения в режиме разработки выполните:
```bash
npm start
```
Приложение будет доступно по адресу `http://localhost:4200/`.

> **Важно:** Для корректной работы приложения (взаимодействия с API без ограничений CORS) рекомендуется использовать браузер **Google Chrome** с флагом --disable-web-security. 
>
> Запустите браузер следующей командой:
> ```bash
> open -n -a "Google Chrome" --args --user-data-dir="/tmp/chrome_dev_test" --disable-web-security
> ```

### Сборка
Для создания релизной сборки:
```bash
npm run build
```
Файлы будут сохранены в директории `dist/`.

---

## 4. Структура проекта

*   `src/app` — основные компоненты приложения.
*   `services/` — сервисы для взаимодействия.
*   `models/` — интерфейсы и модели данных.
*   `proxy.conf.json` — конфигурация прокси для разработки.
